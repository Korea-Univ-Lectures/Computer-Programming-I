포인터 = 주소를 저장하는 변수(0x00000000~0xffffffff) --> 4바이트

모든 포인터 자료형의 크기는 4바이트

포인터 차이 = 앞에 붙는 자료형 = 포인터에 연산했을 때 실제로 바뀌는 값.

int *a // a = 0x11111111

a++; // a : 0x11111115

char *b = 0x11111111

b++; // b : 0x11111112

struct student* a;
